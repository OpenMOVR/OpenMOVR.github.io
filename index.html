Our Vision: Open Science, Eventually Open Data

MOVR is evolving from a traditional registry into an open science initiative — creating transparent, community-driven software tools that accelerate neuromuscular disease research.

We're building the infrastructure for modern research: FHIR-based EMR integration, automated data processing, and collaborative frameworks that bring researchers, patients, and institutions together.

Current Focus (Q2 2026)

  • MOVR 2.0 operational validation
  • Open source tool development
  • Community building via GSoC
  • Transparent governance frameworks

What's Next

  • Clinical data integration libraries
  • Research data access tools
  • Collaborative research platform
  • AI-powered analytics & insights
