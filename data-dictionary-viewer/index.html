MOVR DATA HUB Viewer App (alpha testing)

This tool provides a simple way to view the MOVR Data Dictionary for clinical research. It is designed for both laypersons and open source developers to quickly review the available data fields for Duchenne Muscular Dystrophy (DMD) and Spinal Muscular Atrophy (SMA).

Functionality

The viewer allows you to browse and search the MOVR Data Dictionary. You can filter by disease (DMD or SMA) and see the total number of fields available for each. No advanced features or customization are included in this alpha version.

Basic Metrics

  • DMD Dictionary Fields: 512
  • SMA Dictionary Fields: 412

This app is intended for quick reference and basic review only. Created by Andre Paredes (contact - not-putting-my-real-email.com)
